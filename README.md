# üöÄ Future Wallet: High-Fidelity Financial Projection Engine

**Future Wallet** is a state-of-the-art financial simulation platform built for high-fidelity personal wealth projection. It combines a sophisticated **DAG-based dependency engine** with a premium, **Glassmorphism-styled dashboard** to provide a deep, interactive look into your financial future.

## ‚ú® key Features

### üß† Core Simulation Engine
- **Dynamic DAG Infrastructure:** Model complex financial relationships (Income ‚Üí Tax ‚Üí Savings) using a Directed Acyclic Graph.
- **Monte Carlo Scenarios:** Run deterministic simulations across "Optimistic", "Base Case", and "Pessimistic" trajectories.
- **Shock Modeling:** Simulate random financial crises with rolling recovery slopes and SCD (Shock Clustering Density).

### üí± Multi-Currency & Exchange Dynamics
- **Correlated GBM Dynamics:** Currency rates evolve using Geometric Brownian Motion with mean-reversion.
- **Multi-Wallet Ledger:** Track balances in USD, EUR, GBP, JPY, and more with high-precision arithmetic.
- **Precision Auditing:** Dedicated reports on floating-point drift during high-frequency conversions.

### üè¶ Credit & Taxation Subsystems
- **Credit Score Evolution:** Dynamic scoring based on debt-to-income, punctuality, and restructuring events.
- **Progressive Taxation:** Accurate modeling of income tax, capital gains (ST/LT), and FX-realized gains.

### üé® Premium UI/UX
- **Glassmorphism Design:** Modern, frosted-glass aesthetics with `backdrop-filter` effects.
- **Animated Dashboard:** Interactive Plotly gauges, responsive treemaps, and shimmering hero sections.
- **Recursive Pet State:** A "Financial Vibe" companion that reacts (pulse, shake, bounce) to your simulation's health.

## üöÄ Getting Started

### Prerequisites
- Python 3.10+
- `pip` (Python package manager)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/future-wallet.git
   cd future-wallet
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

### Running the App
Launch the dashboard with:
```bash
streamlit run app.py
```
Open your browser to `http://localhost:8501`.
